# Randomization & Blinding

Defines how participants are assigned to active or placebo and how the blind is maintained across the CP-101 SAD/MAD program.

- **What this governs:** The randomization schedule generation, allocation ratio per cohort, sentinel-dosing rules, and the blinding scheme (who is blinded, how kits map to a code list). Covers emergency unblinding procedures and the firewall: the *blinded* study team stays blinded through study conduct until the final unblinding at lock, while an unblinded firewall — an unblinded statistician and an independent pharmacist — handles assignments and prepares any DSMB / dose-escalation review outputs during the study.
- **Owner:** Unblinded statistician generates and holds the schedule; an independent pharmacist manages kit-to-code mapping. The lead (blinded) biostatistician does not access codes.
- **Connections:** Allocation parameters trace to the [[Protocol]]; the code list drives kit labeling and dispensing under [[Drug Supply & Accountability]]. See the [[CP-101 Knowledge Hub]] for the full note map.

No treatment assignments, seeds, or code lists are stored in this knowledge base.
