# Schedule of Assessments

Defines what is assessed at each CP-101 visit and timepoint, by cohort and dosing day, for both the SAD and MAD phases.

- **Governs:** the visit grid, assessment windows, and timepoint nominal labels that drive collection timing; it does not define results, only when and what is captured.
- **Owner:** Clinical Operations, with Biostatistics and Data Management confirming alignment to analysis needs.
- It is derived directly from the [[Protocol]] and must stay consistent with the [[eCRF Specification]], which fields the same visits and timepoints.
- Sampling timepoints feed the [[PK Sampling Plan]], so any window or timepoint change is reconciled across all three before it is considered final.
