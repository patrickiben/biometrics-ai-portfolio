# Glossary

Single source of truth for CP-101 abbreviations, role labels, and study-specific terms.

- **Governs:** definitions for design terms (SAD, MAD, sentinel dosing, dose escalation), operational acronyms, and any CP-101-specific shorthand used across the knowledge base. Avoids the same term meaning different things in different notes.
- **Owns:** the Lead Biostatistician curates entries; anyone introducing a new acronym in a deliverable should add it here first.
- **Connects to:** terms embedded in file and output names are expanded here, so read alongside [[Naming Conventions]]. Reachable from the [[CP-101 Knowledge Hub]] as the landing point for onboarding.

Keep entries short and unambiguous. When a definition is contested, resolve it here once and reference this note rather than re-explaining inline elsewhere.
