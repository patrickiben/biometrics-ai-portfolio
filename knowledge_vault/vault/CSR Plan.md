# CSR Plan

Defines how CP-101 moves from final TLF outputs to a completed Clinical Study Report.

- **Governs:** CSR authoring approach, statistical contribution timing, in-text table/figure callouts, and the methods/results narrative conventions. Anchors which shells feed the report body versus appendices, and the QC/sign-off path for stat-contributed sections.
- **Owner:** Lead biostatistician drafts statistical methods and results narrative; medical writing owns assembly; the in-house programming team delivers the validated output package on schedule.
- **Connects to:** The [[Statistical Analysis Plan]] is the source of truth for all methods text; CSR tables/figures are produced from the agreed [[TLF Shells]]. Authoring cannot begin in earnest until [[Database Lock]] gates the final outputs. See the [[CP-101 Knowledge Hub]] for the full deliverables map.

Draft outline is staged pre-lock so only numbers, not structure, change after lock.
