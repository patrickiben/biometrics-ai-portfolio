# TLF Shells

Mock tables, listings, and figures for CP-101 — the empty-but-fully-formatted output templates that define titles, structure, footnotes, and population labels with no data populated.

- **Governs:** shell layout, row/column structure, decimal conventions, standard footnotes, and output numbering. Shells are blank by design and carry no reported values.
- **Owner:** biostatistics (shell author) with statistical programming and medical review; currently **Draft** pending SAP finalization.
- **Connections:** every shell traces to a planned analysis in the [[Statistical Analysis Plan]] and is fed by datasets defined in the [[ADaM Specifications]]. Production runs against locked data per the [[Database Lock]] plan, and the resulting outputs flow into the [[CSR Plan]]. The hub index lives in the [[CP-101 Knowledge Hub]].

Shells must be frozen and QC'd before lock so programming targets a stable specification; late footnote or layout edits are logged and re-reviewed.
