The "find hours" decision tree

  1. Reallocate under‑run slack. Move budget from deliverables forecasting under plan (EAC < planned) — the safest source; it is already yours.
  2. Draw the contingency buffer. The planned management reserve exists for exactly this; drawing it is a decision to record, not a silent move.
  3. Defer / re‑sequence lower‑priority in‑scope work. If slack + contingency aren't enough, defer non‑critical-path effort to free hours — with the PM's agreement on the new dates.
  4. Escalate a change order. If a shortfall remains, the work is genuinely additional scope: raise a change order for the shortfall hours rather than absorbing an over‑run silently.
